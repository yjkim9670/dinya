# 포트폴리오 트레이딩 콘솔

한국 주식·ETF·암호화폐를 동시에 추적하는 싱글 페이지 대시보드입니다. 단일 `index.html` 파일로 구성되어 있으며, 차콜 그레이 다크 테마와 스카이 블루 액센트를 활용해 시인성을 높였습니다. 보유 종목 기록은 브라우저 `localStorage`에 저장되어 새로고침 후에도 유지됩니다.

## 핵심 기능

- **포트폴리오 요약**: 총 자산 평가액, 투자 원금, 평가 손익(금액·수익률), 현금 잔고를 한눈에 확인합니다.
- **보유 종목 카드**: 종목별 보유 수량·평균 매수가·현재가·평가 손익·비중·추천 액션을 제공하며, 추천 메시지는 자동으로 계산됩니다.
- **자산군 탭**: 주식(삼성전자·SK하이닉스), ETF(TIGER S&P500·TIGER 나스닥100), 암호화폐(비트코인·이더리움·리플·솔라나)를 탭으로 구분합니다.
- **기술적 분석 지표**: SMA(5/20), RSI(14), 스토캐스틱 Slow(%K/%D), MACD·시그널·히스토그램을 계산하고, 0~100점 추천 점수와 액션 배지를 제공합니다.
- **차트 분석**: Chart.js로 최근 7~90일 가격 추이를 확인하며, 슬라이더로 기간을 조절하고 매수/매도 시그널 포인트를 확인할 수 있습니다.
- **포트폴리오 비중 도넛**: 보유 종목과 현금의 비중을 도넛 차트로 시각화합니다.
- **모바일 대응**: 320px 이상 해상도를 지원하는 반응형 레이아웃과 모바일 전용 햄버거 메뉴(사이드바 토글)를 제공합니다.

## 사용 방법

1. `index.html` 파일을 브라우저에서 직접 열면 대시보드가 실행됩니다.
2. 오른쪽 종목 카드에서 **포트폴리오에 추가** 버튼을 눌러 보유 수량과 평균 매수가를 입력하면 좌측 요약과 도넛 차트가 즉시 갱신됩니다.
3. 이미 보유 중인 종목은 "보유 중 ✓" 배지와 함께 **수정**·**삭제** 버튼이 표시됩니다.
4. **데이터 새로고침** 버튼은 계산과 차트 렌더링을 초기화합니다.

## 포트폴리오 저장 구조

```json
{
  "cash": 10000000,
  "holdings": {
    "005930.KS": {
      "quantity": 100,
      "avgPrice": 70000,
      "addedDate": "2025-01-15"
    },
    "BTC-KRW": {
      "quantity": 0.5,
      "avgPrice": 95000000,
      "addedDate": "2025-02-01"
    }
  }
}
```

## 포지션 추천 로직 요약

- **매수 추가**: 추천 점수 ≥ 80점 && 현재 비중 < 5% → 권장 비중 10%로 증액
- **일부 매도**: 20 ≤ 추천 점수 ≤ 50 && 현재 비중 ≥ 15% → 권장 비중 5%로 감축
- **전량 매도**: 추천 점수 ≤ 20점 → 손절 권장(현재 손익 %) 표시
- **홀드**: 위 조건에 해당하지 않는 경우 포지션 유지

## 개발 메모

- Chart.js CDN만 사용하며, 다른 외부 의존성은 없습니다.
- CSS/JS가 모두 인라인이므로 수정 후 저장하면 즉시 반영됩니다.
- 접근성을 고려해 탭에는 ARIA 속성을 적용했고, 모달은 ESC 키로 닫을 수 있습니다.
- 카드 추가/삭제 시 애니메이션이 적용되어 인터랙션이 자연스럽습니다.
