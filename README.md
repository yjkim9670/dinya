# 포트폴리오 트레이딩 콘솔

모의 데이터를 기반으로 주식·ETF·암호화폐 포지션을 동시에 추적하는 단일 페이지 대시보드입니다. 다크 톤 배경과 청록색 포인트를 사용해 야간에도 눈부심 없이 사용할 수 있으며, 모든 상태는 브라우저 `localStorage`에 저장됩니다.

## 주요 기능

- **요약 카드**: 총 자산 평가액, 투자 원금, 평가 손익, 현금 잔고를 한눈에 확인합니다.
- **보유 종목 리스트**: 비중·평가손익·추천 액션(매수 추가/일부 매도/전량 매도/홀드)을 계산해 배지로 제공합니다.
- **자산군 탭**: 주식, ETF, 암호화폐 세 가지 탭으로 데이터 세트를 전환합니다.
- **기술적 지표**: 단순이동평균(SMA5/20), RSI, 스토캐스틱, MACD를 카드에 요약해 표시합니다.
- **차트 시각화**: Chart.js를 이용한 30일 가격 추세 라인 차트와 도넛 형태의 포트폴리오 비중 차트를 제공합니다.
- **반응형 레이아웃**: 768px 이하 뷰포트에서는 햄버거 버튼으로 사이드바를 토글해 모바일 사용성을 확보했습니다.

## 사용 방법

1. `index.html`을 브라우저에서 직접 열면 즉시 대시보드가 렌더링됩니다.
2. 자산 카드의 **포트폴리오에 추가** 버튼을 눌러 보유 수량과 평균 매수가를 입력합니다.
3. 이미 보유 중인 종목은 "보유 중 ✓" 배지와 함께 **수정**·**삭제** 버튼이 노출됩니다.
4. 입력한 데이터는 자동으로 저장되며, 새로고침 후에도 유지됩니다.

## 데이터 구조

```json
{
  "cash": 10000000,
  "holdings": {
    "000660.KS": {
      "quantity": 12,
      "avgPrice": 200000,
      "addedDate": "2024-03-01T12:00:00.000Z"
    }
  }
}
```

## 파일 구성

```text
.
├── AGENTS.md        # 저장소 편집 지침
├── README.md        # 현재 문서
└── index.html       # 대시보드(HTML, CSS, JavaScript 인라인)
```

## 개발 메모

- 외부 의존성은 Chart.js CDN 하나만 사용합니다.
- 모든 스타일과 스크립트는 `index.html` 내부에 포함되어 있어 배포가 간단합니다.
- 데이터는 실제 API가 아닌 랜덤 변동값으로 생성된 Mock 데이터입니다.
